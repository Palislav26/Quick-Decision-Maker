# Quick Decision Maker
A Streamlit app built as a personal project to help my wife quickly decide on food, activities, and plans without overthinking.

The app gives you **one final decision** â€” no re-rolls.  
If you want a new outcome, you must refresh the page.

---

## Features

- Big, simple button-based flow
- Indoor and Outdoor decision paths
- Random selection from curated options
- Decision is **locked** after choosing
- Background changes based on activity type
- Mobile-friendly and easy to use
